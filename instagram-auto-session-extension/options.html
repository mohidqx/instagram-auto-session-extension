How It Works

  1. Extension detects when you visit Instagram.com
  2. Checks for valid Instagram session in browser cookies
  3. Shows consent page asking for your explicit permission
  4. Only after approval, extracts session data locally
  5. Sends data to your configured Telegram bot
  6. Clears temporary data for security

How to Find Your Chat ID

1

Start a chat with your bot

Send any message to your Telegram bot

2

Get updates from Telegram API

Visit this URL in your browser (replace YOUR_BOT_TOKEN):

https://api.telegram.org/botYOUR_BOT_TOKEN/getUpdates
3

Find your Chat ID

Look for "chat":{"id": in the response. That number is your Chat ID.
